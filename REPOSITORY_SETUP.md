# Repository setup

Repository: `rbt4/pacefold`

1. Open **Settings → Pages**.
2. Set **Source** to **GitHub Actions**.
3. Open **Actions** and confirm “Deploy Pacefold to GitHub Pages” succeeds.
4. Open `https://rbt4.github.io/pacefold/` in Microsoft Edge.
5. Use Edge’s install icon or **Apps → Install Pacefold**.
6. Pin it from `edge://apps`.

Pacefold must remain open or minimized for live JavaScript timers and taskbar badges to update. Workplace policy may disable installation or badges.
