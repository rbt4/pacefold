Privacy

Pacefold does not transmit your activity ledger, rhythm profile, faith selection, location or ergonomic records. The optional OneNote connection sends only captures you write.

Local records

Profile, custom moments, acknowledgements, notification-action history, personal pauses, hydration progress, preparation timers, meals, eye and movement resets, away sessions, captures and an unfinished capture draft are stored in this browser using localStorage. A clicked notification action is held briefly in same-origin cache storage until Pacefold records it, preventing a lost or duplicate click across reloads. These records remain on this device unless browser data is cleared.

Optional Microsoft OneNote connection

OneNote sync is off until the user provides a Microsoft Entra application ID, signs in and chooses a notebook section. Pacefold then requests delegated Notes.ReadWrite through Microsoft identity and sends capture time, category and text to Microsoft Graph. It creates or updates one dated page per day. No client secret or application-only permission is used.

Queued captures remain local when offline, signed out or blocked by workplace policy. Disconnecting keeps local captures. Pacefold never sends hydration, prayer, meal, away, movement or screen-use records to OneNote. See the OneNote setup guide.

Faith-aware profiles

Islamic prayer times are calculated inside the app from the selected coordinates. Other faith-aware presets are editable personal reminders and are not presented as official religious, legal or denominational time rulings. Pacefold does not transmit a selected profile.

Location

Location permission is requested only after the user chooses the Location control. Coordinates are stored locally. Pacefold does not send them to a prayer-time, mapping or analytics service.

Sound

Built-in sound textures are generated locally. A chosen local audio file is represented by a session-only browser object URL. A direct HTTPS audio source is contacted only after the user saves and plays that URL; its host may receive normal network request information.

Other network use

The site and future application updates are downloaded from GitHub Pages. Microsoft identity and Graph are contacted only for the optional OneNote connection. Pacefold does not call analytics, advertising or weather services.

Notifications and taskbar badges

These are optional browser and operating-system features. Permission is requested only after a user presses an enable or test control. Live cues require the app to remain open or minimized and may be disabled by workplace policy.

Display comfort

Comfort preferences affect only Pacefold’s interface. Opening Windows Night light is a user-initiated operating-system settings action; Pacefold does not read or control monitor calibration.

Automatic updates

Pacefold periodically checks its GitHub Pages service worker for a new version. A validated release replaces cached application files automatically while local records remain in browser storage. An update waits rather than reloading over setup or an unfinished capture.
