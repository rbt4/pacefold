Privacy

Pacefold does not collect or transmit your activity record.

Local records

Settings, acknowledgements, prayer mini-breaks, hydration progress, meal windows, timers and away sessions are stored in your browser using localStorage. They remain on this device unless browser data is cleared.

Network use

The initial website and future application updates are downloaded from GitHub Pages. After first load, an offline cache keeps the app available. Pacefold does not call analytics, advertising, location, prayer-time, weather or account APIs.

Notifications and taskbar badges

These are optional browser and operating-system features. Pacefold requests notification permission only after you press an enable or test button. Notifications and live badge changes require the app to remain open or minimized and may be disabled by workplace policy.

Automatic updates

Pacefold periodically checks its own GitHub Pages service worker for a new version. A verified release replaces the cached application automatically.

GitHub Pages

GitHub may process ordinary web-server information under GitHub’s own policies. Pacefold itself includes no tracking scripts or remote data submission.
